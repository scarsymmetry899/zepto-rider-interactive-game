# 🚀 Zepto Delivery Rush — Gamified Experience

## 🎮 Overview

Zepto Delivery Rush is a high-intensity, gamified, single-file HTML experience that simulates the journey of a Zepto delivery agent — from order pickup to final delivery.

It blends:
- Real-time gameplay mechanics  
- Interactive decision-making  
- Humor + Indian context  
- Performance-based outcomes  

---

## 🎯 Core Concept

“Can you complete a Zepto delivery under pressure — fast, safe, and smart?”

---

## 🧩 Experience Flow

1. Boot Splash  
2. Order Card  
3. Grocery Picking  
4. Bike Delivery (core gameplay)  
5. Arrival  
6. Dial Screen  
7. Customer Call  
8. Elevator  
9. Corridor  
10. Final Delivery + Result  

---

## 🛠 Key Features

### Grocery Picking
- 3-shelf layout
- Target vs decoy items
- Zepto paper bag with logo

### Bike Delivery
- Rider with delivery bag
- Tap & hold controls
- Gradual traffic difficulty
- Obstacles: cars, dogs, pedestrians
- Signal system with consequences

### Dial + Call System
- Scrambled keypad
- Swipe-to-call interaction
- Choice-based customer responses
- Mood impacts rating

### Elevator + Corridor
- Funny wrong-floor scenarios
- Door interactions
- Customer appears physically

### Result System
- Star rating
- XP breakdown
- Performance metrics
